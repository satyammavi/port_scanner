# port_scanner
Fast python based port scanner
* python3 portscanner.py -s 1 -e 10000 -V 127.0.0.1
* 
## portScaner
 
 This tools is for fast open port scaner.
 
## Install
```bash
git clone https://github.com/satyammavi/port_scanner.git
cd port_scaner
python3 portscanner.py
```
